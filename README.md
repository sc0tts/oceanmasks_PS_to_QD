# oceanmasks_PS_to_QD

This repository contains exploratory code to examine the relationship between the NSIDC 25km polar stereo grids used in the NOAA sea ice concentration CDR and the OISST quarter-degree lat/lon grid.

The code was developed using a variety of locally installed tools/packages, including:
  - NCO routines (command line)
  - GDAL (on the command line)
  - python (numpy and netCDF4)
  - imagej (manual, not batch mode)

The code is not really designed to be installed or used by anyone but the author.  This is an archive of a preliminary data exploration.

Useful comments are found here in:
- `README_cdr_vs_qd_masks.txt`
- `COMMENTS_OISST_vs_CDRv6_ocean_masks.txt`
